# examples 说明

这个目录只服务一件事：

给总控 agent 和 7 个 stage skill 提供明确样例。

## 子目录

- `中间产物样例/`
  - 用来约束 `source-analysis` 到 `chinese-mirror-pack-translator` 的结构和粒度
- `最终交付样例/`
  - 用来约束最终 8 个交付文件的结构、语言风格和专业程度

## 注意

1. 这些样例是缩略样例，用来展示结构、质感和专业度，不代表完整篇幅。
2. 正式交付时只保留 Word 版本。
3. 样例用 `.md` 和 `.json`，只是为了让 agent 容易读取和比对。
4. 最终 8 个 Word 文件都必须带 `Quick Reference`。
