## 第 01 集

**本集核心事件：** 玛雅买下带编码的怀表、拒绝高价回购，当晚发现卖家身亡，且对方口袋里有针对她的监视照片。  
**本集情绪基调：** 表面冷静、底层被窥视感逐步收紧。  
**本集关键反转：** 危险已从「抽象威胁」落到「她本人已是目标」。  
**本集结尾钩子：** 她拿起一次性联络卡，而不是报警。

## 镜像核对（分镜表）

- `episode_id` / `scene_id` / `shot_id` 与英文 `03` 逐行一致  
- `shot_type` / `rhythm` / `transition` 枚举与英文一致（`transition` 仅用 `CUT` / `HARD_CUT` / `SMASH_CUT` / `MATCH_CUT` / `REACTION_CUT` / `HOLD` / `DISSOLVE` / `FADE_OUT`）  
- `character_ids` / `location_id` / `prop_ids` / `look_ids` 与英文一致  
- 每场首镜为 `environment_establish`（第一或第二镜），末镜为 `hook`  
- `screenplay_ref` 锚点与英文同场次同块对应（可中英混写指针，但不得改顺序）  
- `estimated_duration_sec` 每集累计仍在 60–120s 区间（与英文一致）
