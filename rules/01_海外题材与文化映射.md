海外题材与文化映射规则

来源：
1. `short-drama原始参考/genre-guide.md`

改写目的：
将原有“中文短剧出海参考”改写为适用于“英文网文改编海外短剧”的题材、文化和市场映射规则。

一、适用范围

本规则主要用于：
1. 识别英文原网文的核心题材卖点。
2. 判断其是否适合改造成海外短剧。
3. 把原始设定映射到目标海外市场更熟悉的表达。

二、默认目标市场

如果项目没有特别指定 `target_market`，默认按以下市场处理：

海外英语区主流短剧用户

参考平台心智：
1. ReelShort
2. DramaBox
3. 同类海外短剧 App / 短视频平台

三、已验证的高转化元素

优先级 S：
1. Billionaire Romance
2. Werewolf / Alpha
3. Flash Marriage
4. Secret Baby

优先级 A：
1. Revenge / Payback
2. Hidden Identity
3. Rejected Mate
4. Mafia / Dark Romance

优先级 B：
1. Royal Romance
2. Fake Dating
3. Enemies to Lovers
4. Second Chance

四、题材映射原则

把原文题材改造成海外短剧时，应优先做“卖点映射”，而不是“字面翻译”。

可参考以下映射：

1. 霸总 / 豪门
-> Billionaire / CEO / Heir

2. 战神归来
-> Special Forces / Ex-Military / Decorated Veteran

3. 重生穿越
-> Second Chance / Time Loop / Fate Reset

4. 宫廷权谋
-> Royal Family / Nobility / Dynasty Conflict

5. 萌宝 / 血脉
-> Secret Baby / Hidden Heir

6. 修仙 / 异能
-> Supernatural / Werewolf / Vampire / Dark Fantasy

五、文化场景替换规则

原文或中式改编中常见场景，进入海外版本时需要替换为目标市场更熟悉的场景。

建议优先替换：

1. 宴会 / 家宴
-> Charity Gala / Fundraiser / High Society Banquet

2. 宗族议事
-> Boardroom / Courtroom / Family Trust Meeting

3. 春节团聚
-> Christmas Gathering / Thanksgiving Dinner / Family Reunion

4. 中式校园活动
-> Prom / Graduation / Campus Event

六、文化符号替换规则

建议优先使用以下海外常见符号：

1. Black Card
2. Family Trust
3. Lawyer's Letter
4. Private Jet
5. Yacht
6. Prenup
7. DNA Test

七、行为逻辑改写规则

海外观众更容易接受以下逻辑：
1. 更直接的情绪表达
2. 更快的对抗升级
3. 更明确的利益驱动
4. 更强的个人意志

因此改编时应倾向：
1. 当面对峙优先于背后算计
2. 快速反击优先于长时间忍耐
3. 个人选择优先于家族服从
4. 具体利益动机优先于抽象道德绑架

八、需要规避的中式叙事陷阱

以下内容不应直接进入海外成稿：

1. “你不孝” 作为主冲突
替代：
经济控制、情感操控、继承权打压

2. 跪地道歉
替代：
公开道歉、法律后果、行动补偿

3. 长时间忍辱负重
替代：
缩短压抑周期，更快进入反击

4. 无动机的闺蜜 / 兄弟极端背叛
替代：
加入利益、嫉妒、权力或自保动机

5. 纯苦情哭诉
替代：
压缩苦情段落，快速进入行动

九、目标市场下的人物模型偏好

若按海外英语区主流短剧用户处理，建议优先使用以下人物模型：

1. Powerful but emotionally damaged male lead
2. Misjudged but resilient female lead
3. Public identity vs hidden identity
4. High-status antagonist with visible leverage
5. Child / bloodline / inheritance as pressure trigger

十、接入建议

本规则建议主要接入：

4. `english-screenplay-writer`

十一、使用底线

文化映射的目标不是“硬套欧美标签”，而是：
1. 保留情绪价值
2. 重构叙事容器
3. 提高海外受众的理解与代入效率
