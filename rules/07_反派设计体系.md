反派设计体系

来源：
1. `short-drama原始参考/villain-design.md`

改写目的：
把原有中文微短剧的反派设计法，改造成适用于海外短剧改编的对抗升级工具。

一、核心原则

反派越强，主角越燃。

反派的价值不在于“坏”，而在于：
1. 制造压力
2. 放大主角欲望
3. 提供更高层次的打脸和翻盘空间

二、四层反派结构

1. 小反派
作用：
   - 前期制造日常压迫
   - 快速建立观众情绪债务

2. 中反派
作用：
   - 中前段的主要阶段对手
   - 明显提高冲突强度

3. 大反派
作用：
   - 主线最强对抗对象
   - 与主角有深层利益或命运冲突

4. 隐藏反派
作用：
   - 后期反转
   - 把故事抬到更高层次

三、使用原则

1. 并非所有题材都必须完整四层。
2. 但必须有“威胁升级感”。
3. 每击败一层，应出现更高层压力。
4. 高层反派最好有前置伏笔。

四、反派设计重点

1. 动机必须具体
不要只写“因为坏”。

2. 杠杆必须清晰
反派必须掌握真正能伤害主角的资源、身份或关系。

3. 出场时机必须合理
不能太早透底，也不能突兀空降。

五、不同题材下的反派形式

1. Romance / Billionaire
可用：
   - Rival lover
   - Family manipulator
   - Inheritance controller

2. Revenge
可用：
   - Former betrayer
   - Power holder
   - System-level oppressor

3. Secret Baby / Hidden Heir
可用：
   - Bloodline suppressor
   - Legacy thief
   - Family gatekeeper

4. Werewolf / Dark Fantasy
可用：
   - Alpha rival
   - Clan manipulator
   - Hidden mastermind

六、反派递进禁忌

1. 每层反派强度差不明显
2. 大反派太早被透底
3. 隐藏反派没有前置痕迹
4. 反派被击败后没有更大威胁承接

七、英文剧本中的反派落法

在英文成稿里，反派不应只靠“骂人很凶”体现。

应通过以下方式建立威胁：
1. 资源碾压
2. 信息压制
3. 社会地位压制
4. 情感控制
5. 血缘 / 继承 / 法律杠杆
