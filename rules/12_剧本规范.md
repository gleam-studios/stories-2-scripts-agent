# 12 剧本规范

## 一、适用文件

1. `02_<剧名>_英文剧本.docx`
2. `06_<剧名>_中文剧本.docx`

## 二、Quick Reference 必须包含

1. 项目名
2. Logline
3. 目标市场
4. 类型
5. 集数
6. 单集时长
7. 核心情绪/视觉标签
8. 主角速览

## 三、主体格式

固定格式：

`EPISODE -> SCENE -> INT./EXT. -> 动作 -> 角色名 -> 对白`

### 场景格式

```text
## EPISODE 01

### SCENE 01.01
INT. LOCATION - NIGHT

Action line in present tense.

CHARACTER NAME
Dialogue line.
```

## 四、强制要求

1. 每场都必须有 `scene_id`。
2. 所有 slugline 必须用 `INT./EXT.`。
3. 角色名必须大写。
4. 剧本正文不得缺少动作行和对白。

## 五、禁止项

1. 禁止写 shot size、机位、运镜、转场、AI 备注。
2. 禁止把剧本写成小说叙述。
3. 禁止把设定集当作剧本前言整段复制。
