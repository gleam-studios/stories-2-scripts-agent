# 15 双语镜像规范

## 一、适用文件

1. `05_<剧名>_中文大纲.docx`
2. `06_<剧名>_中文剧本.docx`
3. `07_<剧名>_中文分镜表.docx`
4. `08_<剧名>_中文设定集.docx`

## 二、核心原则

1. 中文 4 份文件只从最终英文 4 份文件生成。
2. 中文 4 份文件必须与英文 4 份文件完全镜像。
3. 镜像的是结构、编号、ID、功能，不是逐字直译。

## 三、必须保持一致的内容

1. 文件类型顺序
2. `episode_id`
3. `scene_id`
4. `shot_id`
5. `character_id`
6. `location_id`
7. `prop_id`
8. `look_id`
9. **分镜表（`07`）与英文分镜（`03`）的机读结构**（见下节）

## 四、分镜表镜像专项（`07` ↔ `03`）

中文分镜表必须与英文分镜表 **逐行对齐**，以满足自动化生产流水线解析。

### 4.1 三级结构一致

每集须含：**集级摘要块** → **场级信息块（8 字段）** → **shot 逐行表**。场次顺序、镜头顺序与英文版完全相同。

### 4.2 表头列名（与英文同源）

`07` 的 Markdown / 表格导出须使用与 `rules/13_分镜表规范.md` 一致的 **21 列**英文 snake_case 列名（与 `storyboard_pack.json` / `schema.json` 同源），顺序为：

`shot_id` · `episode_id` · `scene_id` · `shot_type` · `screenplay_ref` · `visual_content` · `character_action` · `key_info_point` · `shot_language` · `dialogue_anchor` · `emotion` · `rhythm` · `estimated_duration_sec` · `transition` · `sfx_music` · `continuity_notes` · `character_ids` · `location_id` · `prop_ids` · `look_ids` · `ai_note`

### 4.3 枚举与 ID（禁止本地化）

以下字段的值必须与英文版 **逐字一致**（大小写一致）：

- `shot_type`（`environment_establish` / `character_enter` / `action_advance` / `dialogue_delivery` / `emotion_reaction` / `info_reveal` / `reversal` / `hook`）
- `rhythm`（`fast_cut` / `normal` / `pause` / `sustained_pressure`）
- `transition`（`CUT` / `HARD_CUT` / `SMASH_CUT` / `MATCH_CUT` / `REACTION_CUT` / `HOLD` / `DISSOLVE` / `FADE_OUT`）
- 全部 `character_ids` / `location_id` / `prop_ids` / `look_ids`
- `emotion`（与英文版逐字一致，便于双语 diff 与下游情绪标签解析）

### 4.4 允许且应当本地化为中文的单元格内容

- `visual_content` / `character_action` / `key_info_point` / `sfx_music` / `continuity_notes` / `ai_note`
- 集级、场级摘要里的叙事句（若与英文并排展示，语义对齐即可）
- `dialogue_anchor`：使用**中文版对白**；无对白时写 `无`（与英文 `None` 语义对应，流水线需统一映射为无对白）

### 4.5 `screenplay_ref` 与 `shot_language`

- `screenplay_ref`：须能回指到**同一条**英文分镜所锚定的剧本位置；允许保留英文指针格式（如 `S01.01 MAIN BODY block 4`），或在指针后括注中文简述，**不得改变场次或镜头对应关系**。
- `shot_language`：景别/机位/运镜可用英文专业缩写（`WIDE` / `MCU` 等）与英文版保持一致，便于资产与生成管线统一；若团队需要，可在竖线后追加中文构图说明，但**不得删改**与英文版一致的核心镜头术语段。

### 4.6 禁止项（分镜层补充）

1. 禁止增删任何 `shot` 行或调换顺序。
2. 禁止更改任何 `shot_id` / `scene_id` / `episode_id`。
3. 禁止把 `transition` 写成自由中文（如「硬切」「叠化」）替代枚举值。
4. 禁止省略 `sfx_music` 或 `continuity_notes`（与 `rules/13` 一致，全列必填）。

## 五、允许变化的内容

1. 语言表达
2. 中文人名译名
3. 中文术语顺口化

但这些变化不能破坏任何结构对应关系。

## 六、alignment notes 最低要求

1. 翻译策略说明
2. 人名对照表
3. 术语处理说明
4. 镜像核对清单
5. **分镜表**：`07` 与 `03` 逐行 `shot_id` 对照表（或勾选清单），确认 21 列齐全且枚举未漂移

## 七、禁止项

1. 禁止回到原始英文网文重新生成中文版本。
2. 禁止增删场次、镜头、设定块。
3. 禁止更改任何 ID。
4. 禁止把中文版本当作二次创作版本。
5. 禁止在中文分镜表中改用与 `rules/13` 不一致的列名、列序或 `transition` 自由文本
