开场规则

来源：
1. `short-drama原始参考/opening-rules.md`

改写目的：
将原有面向中文微短剧的开篇规则，改写成适用于海外短剧英文成稿的开场约束。

一、核心原则

前 5 秒定生死。

开场不是介绍世界观的地方，而是建立抓手的地方。

第一场必须至少完成以下任务中的两项：
1. 建立核心关系
2. 抛出直接冲突
3. 制造极端处境
4. 提出核心问题
5. 释放第一层人物反差

二、禁止项

开场禁止以下写法：

1. 慢节奏环境空镜
2. 大段背景说明
3. 没有冲突的闲聊
4. 解释设定式旁白
5. 三段之后还没有出现钩子

三、推荐的 6 类开场模板

1. 直接打脸型
适用：
   - Revenge
   - Hidden Identity
   - Power fantasy

开场动作：
1. 主角被羞辱 / 被看低
2. 高压不公立刻出现
3. 观众立刻等待反击

2. 身份反转型
适用：
   - Billionaire
   - Hidden Heir
   - Secret Identity

开场动作：
1. 先制造“低位误判”
2. 再快速抛出真正身份线索

3. 极端困境型
适用：
   - Flash Marriage
   - Secret Baby
   - Dark Romance

开场动作：
1. 主角陷入不可回避危机
2. 立刻做出高风险选择

4. 倒叙悬念型
适用：
   - Mystery
   - Betrayal
   - Second Chance

开场动作：
1. 先给出高能结果
2. 再切回前因，制造追问

5. 强情绪吸引型
适用：
   - Romance
   - Enemies to Lovers
   - Emotional melodrama

开场动作：
1. 给出强情绪碰撞
2. 让观众先被关系吸住

6. 第二次机会型
适用：
   - Second Chance
   - Time Loop
   - Reset destiny

开场动作：
1. 主角明确知道“这次不一样”
2. 立刻进入行动

四、开场必须具备的钩子

第一场必须在结尾前明确形成至少一种钩子：
1. 悬念钩
2. 身份钩
3. 危机钩
4. 情绪钩

五、英文成稿时的开场要求

英文开场必须做到：
1. 快
2. 直
3. 有功能

具体要求：
1. 动作句短
2. 对白直达冲突
3. 不使用小说式慢描写
4. 第一场内完成“抓住观众”的目的

六、开场自检清单

写完开场后至少检查：

1. 前三段内是否出现冲突或强问题？
2. 主角关系是否已建立？
3. 第一场结尾是否有继续看的理由？
4. 是否存在不必要的背景说明？
5. 是否更像网文开头而不是短剧开头？

七、接入建议

本规则建议接入：

2. `english-outline-writer`
