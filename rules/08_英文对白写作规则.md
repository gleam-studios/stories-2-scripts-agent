# 08 英文对白写作规则

## 一、定位

对白是场次的**情绪与冲突引擎**，不是叙事摘要工具。

每句对白只做三件事之一：**推进冲突 / 揭露信息 / 改变情绪**。不满足任何一条的对白行，删掉或合并。

---

## 二、七条核心原则

### 2.1 短句优先

高压对抗场中，一句对白超过 15 个词就要拆。节奏靠短句堆出来，不靠长句讲出来。

### 2.2 冲突优先

对白首先服务于对抗、诱惑、揭露和关系变化。解释型对白（角色替剧情做注脚）是最常见的错误。

### 2.3 功能性写作

每句对白必须做到以下三种功能之一：

| 功能 | 说明 | 判断方式 |
|------|------|---------|
| **推进冲突** | 对抗态势改变，局面更紧或出现意外 | 读完这句，另一角色的处境是否不同了？ |
| **揭露信息** | 给出此前不知道的事 | 读完这句，观众或另一角色知道了什么新东西？ |
| **改变情绪** | 场内情绪方向转换 | 读完这句，这场戏的情绪基调有没有偏移？ |

### 2.4 口语优先

英文对白的核心口语标志：

- 缩略形式：`I'm` / `don't` / `can't` / `won't` / `it's` / `you've`
- 填充词（用于紧张或思考时）：`look` / `wait` / `hey` / `listen`
- 句尾确认词：`right?` / `got it?` / `understand?`
- 不完整句：`Fine.` / `Not now.` / `Enough.`

### 2.5 角色声音必须可区分

每个主要角色必须有稳定的语言注册（linguistic register）。同一场里，不同角色的对白不能读起来像一个人在说话。

**判断方法：** 遮住角色名，只看对白，能否判断出是谁在说——如果不能，对白需要重写。

### 2.6 少解释，多碰撞

不要让角色替作者写说明书。信息通过碰撞中的反应来揭露，不通过独白来解释。

超过 3 句同一角色连续对白（中间无动作行打断）= 解释型独白，必须拆散。

### 2.7 留白与潜台词

不是每句话都要把意思说满。压力最大的时刻，往往是说得最少的时候。

## 三、角色声音区分规则

每个主要角色在对白层面需要有至少 2 个固定特征，全季保持一致：

| 特征维度 | 说明 |
|---------|------|
| 句长倾向 | 长句/短句 |
| 句式倾向 | 陈述/反问/命令/省略 |
| 用词层次 | 正式/口语/行业黑话/简单词 |
| 情绪表达方式 | 直接说出/用反话/沉默/反击 |

### 示例：三类角色的声音区分

**角色 A（主角，被压制的一方）：**
- 短句、口语、句子不完整
- 情绪在动作行里，台词尽量克制
```
MAYA
Fine.

MAYA
I heard you.

MAYA
(looks away)
It doesn't matter.
```

**角色 B（强势对手/高位者）：**
- 句子完整，不省略
- 从不急着说话，用问句控制节奏
- 信息给得少，但每句都精准
```
ELLIOT
You already know what I'm going to say.

ELLIOT
How long have you had it?

ELLIOT
I'll wait.
```

**角色 C（中间人/信息掌握者）：**
- 句子中等长度
- 习惯用条件句，不给直接答案
- 给信息，但总是不完整
```
REED
That depends on what you're willing to give up.

REED
I can get you close. Not all the way.

REED
Ask me again in an hour.
```

**反面示例（三角色对白无法区分）：**
```
MAYA
I need to know the truth. What are you hiding from me?

ELLIOT
I am telling you the truth. You need to trust me on this.

REED
Both of you are overreacting. Let's think about this carefully.
```
— 三人说话节奏、句长、句式完全相同，角色消失了。

## 四、场景分类对白规则

### 4.1 高压冲突场

- 单句上限：15 个词
- 禁止一口气解释（超过 3 句无打断 = 独白）
- 反问优于陈述（`You knew?` 比 `I can't believe you knew about this.` 更有力）
- 角色之间抢话、打断对方，是正常的

```
NORA
You set me up.

CELIA
I protected this family.

NORA
By selling me out.

CELIA
By making a choice you were too weak to make.
```

### 4.2 情绪爆发场

- 可稍长，但不能变成演讲（不能超过 4 句无打断）
- 情绪转折点不能只靠对白说出来——必须有动作行标注（角色站起/摔东西/转身）
- 高强度情绪落点处，对白反而要短

```
NORA
You want to know what I remember? I remember exactly which day you stopped calling me your daughter.

She puts the photograph face-down on the table.

NORA (CONT'D)
That was enough.
```

### 4.3 解释性信息场

- 把信息拆成多轮 Q&A，不要单角色一口讲完
- 接收方的反应行（包括沉默）是独立动作块，不能省
- 接收方的追问要体现他/她的知识状态（不要提问得太全知）

```
ELLIOT
The engraving is a routing key.

MAYA
For what?

ELLIOT
An account. Dormant for eleven years.

MAYA
How much?

ELLIOT
Enough that three people are already dead.
```

### 4.4 决裂场

- 对白更短、更硬，拒绝解释
- 结尾一句落地即停，不回头

```
NORA
I'm done.

CELIA
You can't just—

NORA
Watch me.
```

### 4.5 告白/情感揭露场

- 直接，不绕弯子，不用比喻代替表达
- 被说的角色必须有动作反应行（不能只有台词回应）
- 不要双方同时说出情绪——留一个沉默、一个说

```
MAYA
I trusted you. That's all this was.

Elliot doesn't answer. He picks up his jacket.

MAYA (CONT'D)
Don't.

He stops.
```

### 4.6 身份揭露场

- 接收方第一句话应该是震惊/否认/反击，不是平静接受
- 揭露方的台词简洁，留出观众反应空间
- 揭露之后必须有沉默动作行，不能立刻追加解释

```
ELLIOT
Your name isn't Maya Sterling.

A beat.

MAYA
(very still)
What did you just say?

ELLIOT
You heard me.
```

## 五、常见错误与修正

| 错误类型 | 错误示例 | 修正方向 |
|---------|---------|---------|
| 中式直译腔 | `"I have already tolerated you for a very long time."` | `"I'm done tolerating you."` |
| 信息堆积独白 | `"The watch contains routing credentials that can unlock a dormant cartel account worth nine hundred million dollars."` | 拆成多轮，让对方追问 |
| 情绪直说 | `"I feel so betrayed and hurt right now."` | 写进动作行；台词只说行动 |
| 全知追问 | `"Who told you? What did they say? When did this happen?"` | 角色只问最急的一个问题 |
| 书面句式 | `"It is imperative that you understand the gravity of the situation."` | `"You don't understand what you're holding."` |
| 角色无区分 | 所有角色的台词节奏、句长相同 | 每个主要角色至少 2 个固定语言特征 |

---

## 六、禁止项

1. 禁止中式英语直译腔（见上方错误示例）
2. 禁止长篇解释型独白（同一角色连续 3 句以上，无动作行打断）
3. 禁止书面作文式对白
4. 禁止单轮对白里塞超过 1 个信息点
5. 禁止所有角色说话节奏、句式相同
6. 禁止台词直接说出刚发生过的动作行内容（重复已知信息）
7. 禁止 Parenthetical 解释情绪（情绪写进动作行）
8. 禁止全知追问（一口气问完所有问题）
