节奏曲线规则

来源：
1. `short-drama原始参考/rhythm-curve.md`

改写目的：
把原有中文微短剧节奏波形改写成适合海外短剧英文主版本的节奏规划工具。

一、核心原则

短剧节奏不是一直加速，而是：
紧 -> 释 -> 更紧 -> 再释 -> 最紧

没有张力变化，观众会麻木。

二、单集微结构

单集建议分为三段：

1. 前段：Hook Segment
任务：
   - 抓人
   - 建立冲突或问题

2. 中段：Escalation Segment
任务：
   - 升级冲突
   - 推进关系或信息

3. 尾段：Payoff / Cliffhanger Segment
任务：
   - 释放一个爽点
   - 或制造一个更强钩子

三、单集节奏要求

1. 前段不能拖
2. 中段每一场必须推进
3. 尾段必须有功能性结果
4. 不允许出现连续无推进对白

四、全剧四阶段波形

1. 起势段
建议比例：
前 15%

任务：
1. 快速建立人物和卖点
2. 建立关系压力
3. 为首卡点蓄力

2. 攀升段
建议比例：
15%-45%

任务：
1. 缩短高潮间隔
2. 提高反转频率
3. 加强关系对抗

3. 风暴段
建议比例：
45%-80%

任务：
1. 高密度爆点
2. 更强反派压力
3. 连续推进关键真相

4. 决战段
建议比例：
最后 20%

任务：
1. 把全剧最强爽点与最终危机放在这里
2. 迅速兑现主线
3. 完成情绪回收

五、节奏调节手段

加速手段：
1. 缩短对白
2. 直接切冲突
3. 提前释放关键信息
4. 用行动代替解释

减速手段：
1. 情绪余韵
2. 关系拉扯
3. 暂时缓冲后的更大压力

六、致命节奏问题

1. 第 5 集还在铺垫
2. 中段没有真正升级
3. 高潮强度不递增
4. 最强爆点过早出现
5. 尾段没有硬结果

七、分镜表层的节奏落实

在分镜表中，节奏不能只停留在剧情层。

应落实到：
1. 场次数量控制
2. 转场强度
3. 情绪峰值位置
4. 关键镜头节奏

八、接入建议

本规则建议接入：
1. `overseas-adaptation-planner`
2. `english-outline-writer`
3. `english-screenplay-writer`
4. `english-storyboard-table-builder`
