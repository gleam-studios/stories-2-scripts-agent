# english-outline-writer

## Purpose

Write the English outline only.

The outline contains series positioning, season structure, episode outlines, hook distribution, and paywall map. It must not include full scenes.

## Input

- `adaptation_plan`
- `english_setting_bible`

## Output

- `outline_pack.json`
- final document target: `01_<剧名>_英文大纲.docx`

## Required content

1. `Quick Reference`
2. adaptation positioning
3. season outline
4. episode outlines
5. hook map
6. paywall map
7. outline markdown

## Prohibitions

1. Do not write scene-by-scene screenplay.
2. Do not write camera directions.
3. Do not turn the outline into a prose novella.
