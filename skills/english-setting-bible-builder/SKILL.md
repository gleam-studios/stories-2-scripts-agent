# english-setting-bible-builder

## Purpose

Build the English setting bible.

This document is a heavy design bible. It carries world background, timeline, characters, looks, locations, props, visual system, continuity rules, and AI asset notes.

## Input

- `story_bible`
- `adaptation_plan`

## Output

- `setting_bible.json`
- final document target: `04_<剧名>_英文设定集.docx`

## Required content

1. `Quick Reference`
2. project positioning and world background
3. timeline and backstory
4. detailed character profiles
5. character look design
6. relationship notes
7. location settings
8. prop settings
9. wardrobe and visual system
10. continuity rules
11. AI asset matrix
12. setting bible markdown

## Prohibitions

1. Do not write episode outlines here.
2. Do not write screenplay scenes here.
3. Do not write shot rows here.
